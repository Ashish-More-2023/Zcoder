## ğŸš€ Getting Started

### Prerequisites
- Node.js (v16 or higher)
- MongoDB
- Git

### Installation

```bash
# Clone the repository
git clone https://github.com/Ashish-More-2023/ZCoder.git

# Navigate to project directory
cd ZCoder

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env
```
